Skills

Front End

Tengo habilidades avanzadas en el desarrollo de HTML, lo que me permite crear sitios web y aplicaciones web dinámicas y atractivas. Soy capaz de escribir código HTML limpio y semántico que cumple con los estándares de la web y se adapta a los requisitos de accesibilidad y usabilidad. Además, tengo experiencia en la implementación de estilos y efectos visuales utilizando hojas de estilo CSS, lo que me permite crear diseños y interfaces de usuario atractivos y funcionales. También tengo experiencia en la integración de elementos interactivos y multimedia, como videos, animaciones y efectos de sonido, para crear experiencias de usuario envolventes y atractivas. En resumen, mi habilidad en el desarrollo de HTML me permite crear sitios web y aplicaciones web de alta calidad que cumplen con las necesidades y expectativas de los usuarios y los clientes.

Back End

Mi nivel de conocimiento en SQL es básico, pero tengo algunas habilidades en este lenguaje de consulta de bases de datos. Soy capaz de escribir consultas simples para recuperar datos de una base de datos, utilizando cláusulas SELECT, FROM y WHERE para filtrar y ordenar los resultados. Además, tengo experiencia en la manipulación de datos utilizando operaciones básicas como INSERT, UPDATE y DELETE. Sin embargo, reconozco que mi conocimiento en SQL es limitado y que todavía tengo mucho que aprender para poder utilizar este lenguaje de manera más avanzada. Por ejemplo, aún no tengo habilidades en la creación de tablas y la definición de relaciones entre ellas, ni en la utilización de funciones avanzadas como JOIN y GROUP BY. Sin embargo, estoy dispuesto a aprender y a mejorar mis habilidades en SQL para poder utilizarlo de manera más efectiva en mi trabajo.
